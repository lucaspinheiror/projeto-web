# Execução do projeto 

- 1 Clone o repositório
```bash
git clone https://github.com/lucaspinheiror/projeto-web.git
```
- 2 Entre na pasta do back-end
```bash
cd nest-app
``` 
- 3 Instale as dependências
```bash
npm install
```
- 4 Execute o servidor
```bash
npm run star:dev
```
